Brainsick™ code compiler

About

The Brainsick™ code compiler is a compiler for the BASIC programming language, written in Haskell. It is in the very early stages of functionality.

Brainsick™ is free software.

Downloads and getting started

Please follow the getting started guide, which covers binary downloads.

The source code repository can be browsed online.

FAQ

What’s up with the name “Brainsick™”?

Contrary to popular belief, this does not refer to the state of mind required to write a BASIC compiler. Instead, the etymology is very similar to that of Samba.
