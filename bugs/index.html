Reporting bugs

So, you want to report a bug in bscc, the Brainsick™ code compiler? Well, I have news for you: It is written in a programming language called Haskell. Haskell is statically typed, which means—if it compiles—there are no bugs. (For those who are interested, bscc is the unique non-trivial program with the signature bscc :: [FilePath] -> IO FilePath.)

Ducks.

More seriously, the compiler is in too early a state for your bug report to likely be useful to the developer. Sorry. Though if you really, really, really want to get in touch, you should be able to find contact details inside the source code.
